conference_web

CSS Conference Application Web & Server Implementation

#Requirements

  • PHP 7.x
  • Composer

#How To Run

  1. Navigate to the project directory.
  2. Run the Command: 
    composer install
  3. Run the Command: 
    php artisan serve

#API End-Points Click Here for API End Points

#Database Migrations

##Creating Database Migration

  1. Navigate to the root folder of the project.
  2. Run the Command:
    php artisan make:migration name_of_migration
  3. Note the creation of the file in the ./database/migrations folder

##Setting Up Database Migration When in doubt View Documentation RTFM

###Up Function The Up function is basically where you make modifications to the database. For example, where you put all the code to create your tables

###Down Function The Down function is basically an undo function, where you place code to revert/rollback a migration.

  • Conference name

  • Location
  • Client ID
  • Start Date (will include time)
  • End Date (will include time)

Speakers

  • Speaker Tel Number

Rooms
